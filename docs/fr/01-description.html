XtrmAddons WordPress Widget To Top

Description

Le Plugins WP Xtrm Button To Top est un plugins pour WordPress.
Il ajoute un bouton de retour en haut de la page paramétrable.

Les caractéristiques du Plugin WP Xtrm Button To Top

Le Plugin WP Xtrm Button To Top, c'est:

  • Personnaliser la position du bouton.
  • Personnaliser les dimensions du bouton.
  • Personnaliser l'apparence du bouton.
