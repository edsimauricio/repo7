Introduction

Le Widget Xtrm ToTop est un plugins pour WordPress qui ajoute un bouton de retour en haut de la page paramétrable.

Les caractéristiques du Widget Xtrm ToTop

  • Personnaliser la position du bouton.
  • Personnaliser les dimensions du bouton.
  • Personnaliser l'apparence du bouton.
