XtrmAddons WordPress Widget To Top

Description

The WP Plugins Xtrm To Top Button is a plugin for WordPress.
It adds up back button of customizable page.

The Plugin WP Xtrm Button To Top caracteristics

Plugin WP Xtrm Button To Top, it's:

  • Customize button position.
  • Customize button dimensions.
  • Customize the appearance of the button.
Notice that the screenshot is in french but the plugin is available in english language.
